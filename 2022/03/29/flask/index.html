flask-migrate

flask中一般使用SQLAlchemy框架操作数据库，db.create_all()创建数据库，在创建数据库后若有对数据库结构进行修改的需求，则需要将原数据库进行删除再创建，显然这不是我们期望的。flask-migrate就提供数据迁移的功能，能够随时更新数据库字段

数据库操作

初始化 python manage.py db init

只有第一次需要进行初始化，初始化后会增加一个migrations文件夹，该文件夹下的versions文件夹存放着数据库的版本，此时创建了迁移仓库。

迁移 python manage.py db migrate

生成迁移文件，此时并未创建表

更新 python manage.py db upgrade

更新数据库，创建相应的表

帮助 python manage.py db --help
